Paper Planes - A scalable distributed messaging app

This is a distributed messaging app written in Java, using WebSockets, Spring, ActiveMQ and JavaFX
This was made over roughly a month in a team of 3, and we really tried to push ourselves to create a scalable design
The created frontend is a JavaFX implementation though the backend is independent and could support any front end.
Of my projects to date this is definitely one of the ones i'm most proud of.

Cluedo - Java Swing Boardgame

An older project but still a fun one.
This Recreation of the Cluedo boardgame allows up to 6 players to play Cluedo on their computer,
